# TypeFocus

## A tool to bring more equity to typing
TypeFocus is an accessible typing software for windows that magnifies the area around the caret.
It works system wide save for system programs.

### Hotkeys
- Shift + Alt + M
- - Hide window
- Shift + Alt + N
- - Invert colors
- Shift + Alt + K
- - Decrease zoom
- Shift + Alt + L
- - Increase zoom
- Shift + Alt + P
- - Exit program
